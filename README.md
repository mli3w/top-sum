## 🧠 What is this?

Top Sum is an experiment in **learning through interaction**, not repetition.

Instead of solving math on paper, you:

* Pilot a plane through a 3D world
* Steer toward the correct answer
* Build streaks and enter **flow states**

It’s less “quiz” and more:

> **stay in control, stay in flow, don’t break the streak**

---

## ⚡ Core Gameplay

* Fly through the **correct answer gates**
* Wrong answers or collisions cost lives
* Chain correct answers to build **Flow Multipliers**
* Hit streak milestones → trigger **slow-motion “wow” moments**

---

## 🔥 Features

* 🛩️ Real-time 3D gameplay powered by **Three.js**
* 🧮 Progressive difficulty (addition → subtraction → multiplication)
* 🔁 Flow system (streak-based multipliers)
* ✨ Slow-motion rewards + visual feedback
* 🌆 Stylised Marina Bay environment
* 🌗 Day / sunset mode with dynamic lighting
* 🚤 Moving boats + living world details
* 🎮 Works on:

  * Keyboard (desktop)
  * Joystick / D-pad (mobile)
* 💾 Local best score tracking

---

## 🎯 Why this exists

Most math tools focus on:

> repetition → correctness

Top Sum explores:

> **movement → feedback → engagement → learning**

The goal is to make learning feel like:

* maintaining control
* reacting under pressure
* staying in a flow state
